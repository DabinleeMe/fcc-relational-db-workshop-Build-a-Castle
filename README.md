# 🏰 Build a Castle Workshop

This is a practice project from the **freeCodeCamp: Relational Database Certification**.  
I learned how to navigate the terminal and use the **Nano text editor** to create and manage files and directories.

## 🛠 Skills Practiced
* Terminal Navigation (`cd`, `ls`, `mkdir`)
* File Manipulation (`touch`, `cp`, `mv`, `rm`)
* Text Editing with **Nano** (`Ctrl+O`, `Ctrl+X`)
* Basic Linux Permissions
